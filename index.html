Обзор

Как выбрать синтезатор и другие
клавишные инструменты

Синтезатор стоит выбирать не по количеству тембров, а по сценарию использования.

С чего начать выбор

Под одним названием часто смешивают несколько разных классов. Цифровое пианино ближе к классической технике, MIDI — не заменяет самостоятельный инструмент, а рабочая станция — совершенно другая задача.

Критерии выбора

1

Клавиатура

Количество клавиш, тип механики, чувствительность к нажатию. Для обучения критична молоточковая.

2

Динамики

Встроенные — для дома. Без динамиков — для сцены (нужен монитор).

3

Аккомпанемент

Есть — для любителей. Нет — для сценических и студийных.

Правильный выбор начинается не с бренда, а с понимания, какой из шести типов вам нужен.
